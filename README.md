## ATM Simulator 
I made this ATM simulator in python using customTkinter as GUI and a real fingerprint scanning device for registeration and authorization. <br>
To run it, install the dependendies using this command <br>
```
pip install -r requirements.txt
```
<br>
And then run the main python file using this commmand 
<br>

```
python main.pyw
```
